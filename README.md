## Version Info 

* Webpack: ~4.28.4
* NodeJS: 10.16.0

* TypeScript: 3.9.3
<br />

## Entering dev mode 

1. Install package with command **[yarn]** and **[yarn.lock]** file
2.
```
$ yarn start
```
3. **into 'localhost:8090'**
<br />

## Build project (release)

build project with webpack

```
$ yarn build
```
